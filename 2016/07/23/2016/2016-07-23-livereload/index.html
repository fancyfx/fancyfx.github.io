什么是liveReload？

livereload是一个web开发辅助工具，当我们修改完html、css和js的时候会自动刷新浏览器，不需要再F5了。这样在双屏切图、写js代码的时候会提高很多效率；

就是说有了这玩意你不用手动刷新了，是不是很爽？

如何使用

由于我的开发机是 mac 所以以下操作都是基于mac os

安装

注意：由于我使用的是 npm进行安装 且需要配合Chrome插件进行使用 因此如果你没有安装node.js和无法翻墙安装Chrome插件 下面的操作你可能无法生效

1.使用如下命令安装 LiveReload

2.安装 Chrome 插件 下载地址

使用

1.由于Chrome livereload插件 无法识别flie 协议 因此在本地环境需要配置http服务器

这里简单说一下 mac os 下的 Apache http服务器的使用(Mac OS下自带了apache 因此可以直接简单配置即可)

  • 在终端输入 open /etc/apache2 找到httpd.conf 打开编辑
  • 找到这个DocumentRoot，修改成你想要的地址即可。 这个地址就是你部署项目的文件夹， 你的项目文件必须放到你配置的这个文件夹里面才可以用 http://localhost 访问

关于mac 下Apache 详细配置 请看这里

2.配置好了http服务 在命令行打开 项目目录 执行 livereload

3.然后在 Chrome 中点击插件图标启动监听就可以了
注意：点一下 livereload插件 中心圆点变实心就是启动成功了
未启动 启动成功

其他

如果你觉得上面的操作很麻烦你也可以使用图形化的桌面应用LiveReload.app ，这款应用同时有 Mac 版和 Windows 版，使用起来也很简单，通过图形界面设置好需要监听文件所在的文件夹，然后将一段脚本插入到 HTML 页面即可
